# このテンプレートの使い方

- **ファイル**: リポジトリ直下の `articles/` に `*.md` を置く。Zenn はこのフォルダだけを同期する。
- **新規記事**: このファイルをコピーし、ファイル名（＝スラッグ）を 12〜50 文字の英小文字・数字・ハイフン・アンダースコアで付ける。例: `my-article-slug.md`。
- **Front Matter**: 先頭の `---` で囲んだメタ情報を編集する。`published: true` にすると Zenn に公開される。
- **本文**: 下記の「よく使う記法サンプル」を参考に執筆する。不要なサンプル段落は削除してよい。

## 記事の推奨構成

1. **見出し1（#）** … 記事タイトル（1本に1つ）
2. **見出し2（##）** … 大見出し（セクション）
3. **見出し3（###）** … 小見出し
4. 必要に応じて **見出し4（####）** まで使用

# よく使う記法サンプル

## 見出し

```markdown
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
```

## リスト

**箇条書き（`-` または `*`）**

- 項目1
- 項目2
  - ネスト
- 項目3

**番号付き**

1. 最初
2. 次
3. 最後

## リンク・インライン

- [リンクテキスト](https://example.com)
- **太字** と *イタリック*
- インラインで `code` を書く

## テーブル

| 列A   | 列B   | 列C   |
| ----- | ----- | ----- |
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

## コードブロック

**言語指定（シンタックスハイライト）**

```ts
const greet = (name: string) => `Hello, ${name}!`;
```

**ファイル名付き**（`言語:ファイル名`）

```ts:lib/utils.ts
export const add = (a: number, b: number) => a + b;
```

**diff**

```diff js
+ const added = true;
- const removed = false;
```

## 引用

> 引用文の一行目
> 引用文の二行目

## 区切り線

`---` だけの行で水平線。

---

## メッセージボックス（Zenn独自）

:::message
通常のメッセージ
:::

:::message alert
警告メッセージ
:::

## アコーディオン（Zenn独自）

:::details クリックで開閉
折りたたまれた内容をここに書く。
:::

## 画像

```markdown
![Altテキスト](https://画像のURL)
*キャプション（画像の直下行に * で囲む）*
```

幅指定: `![](https://画像のURL =250x)`

## その他

- **インラインコメント（非公開メモ）**: `<!-- TODO: 追記する -->` → 公開ページには表示されない。
- **URLだけの行**: 前後に空行を入れると Zenn がカード表示する。
- **数式**: ブロックは `$$ ... $$`、インラインは `$...$`（KaTeX）。
